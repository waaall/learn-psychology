随着对人际交往过程中奇怪行为(无论是同学、同事还是亲戚父母、对象、我自己)的疑惑和好奇，实在按捺不住我想要系统化的了解心理学、精神、大脑的相关机理。

自大学以来，我对科技的兴趣只有物理和计算机，所以对此类事物的了解还停留在中学的垃圾应试阶段（几乎等于一无所知），由于我第一份工作是医疗相关，所以强行让我对医学比如免疫、大脑、心脏有了一些浅显但是相对系统化的了解。

当我对心理学相关感兴趣时，一个最大的兴趣点也是我认为人类最终的方向就是如何唯物的解释这些所谓的精神问题。从神经科学和心理学的初期（100-200年前？）人类甚至不知道大脑是精神的物质基础，如今已经进步很多了，但还只是停留在脑区MRI检测整体活跃度的阶段，远远达不到可以定性解释的地步。

正如量子力学的出现，就可以从物理学出发解释化学的最基础原理，比如元素周期表，化学能相关的比如键能等也可以通过量子力学定性的解释，后来演化出量子化学的课程。在此之前，物理和化学的基础知识几乎是隔离的。正如现在的脑科学/神经科学和心理学一样，虽然我们知道脑是精神的物质基础，但是远远没有像物理化学那样被量子力学被发现后的恍然大悟感。我肯定是不能对此有什么贡献了，但是无比的好奇。

在此之前，我还是要了解一下精神分析的方法和近三十年的新理论，这对我了解诸多社交现象有很大帮助。这个的导火索是我发现我妈有narcissistic personality disorder，我经历了恐惧/怜悯/厌恶/无奈等众多复杂负面情绪后，还有对其的好奇。我在研究NPD时接触了一些心理动力学理论，其实蛮有意思的，以至于我忍不住暂时搁置我的“主线”（计算机领域工作）来研究这个。甚至在此过程中我发现自己有较为严重的ADHD（其实只有注意力缺陷没有多动），忽然明白了我为何几十年来都对自己不感兴趣的领域或者枯燥的事儿及其的反感而毫无动力极其拖延，感兴趣的就很上头；高中什么也不想学就想辍学，语文不及格上大学却喜欢看小说各种逃课看小说，研究生不想水论文直接休学，极度依赖提醒事项等等等等。了解这个并没有对我的病情有缓解，但一是对我的自我厌恶感有很大的改善，二是对我的不理解感有很大的满足。

对，我无法控制自己所好奇事物整体框架的研究，我现在认为这绝对是一个比较严重的心理问题，虽然它对我有好处，但是很困扰。
比如我在研究生期间由于研究的是物理问题，但是我基础不足，在我对物理有整体的了解之前---我的理解到了一定的细节程度，且能构建起我对这个领域的一颗稳定的“知识树”。我能清楚的知道这个树根基不稳或者完全无法连接在一起的感觉，怀着这种感觉我是无法进行细枝末节的研究工作的。表现也就是我无法进行不在树上的工作，如果非要我干，那么AD就出来控制我了。
再比如我在研究生的第二年开始对计算机感兴趣之前，我打字都学不下去。然而在我对计算机的操作系统、硬件的基本原理、编译的机器指令是如何运行这些问题没有框架和部分细节的理解之前，我是不想学习任何一门编程语言的。若让我在这些问题解决之前做到精通这些并做出一个及格的“作品”，那我quit。

这些可能会在多年后带给我好处，但我清楚，这些带来好处之前坏处已经够多了，而且我并不多么享受这些好处，而更享受在大脑中“种出一颗树”的感觉。这很不正常，我还不确定这是什么心理问题。我认为这些现象最终会被神经科学解释，类似做一个详细的MRI&脑电分析后就可以定量给出一个人各种精神问题。想到这里就无比兴奋。哪怕可能把我“种的这颗心理学树”连根拔起。

Anyway，这是我开始种这颗心理/精神知识树的笔记仓库。
