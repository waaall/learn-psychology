
## MBTI 不是严谨的科学但也不是星座


### 理解 MBTI 的分类方式

- I / E（内向 / 外向）
- S / N（感觉 / 直觉）
- T / F（思维 / 情感）
- J / P（判断 / 感知）

如果仅凭这几个字母，或者网络上流行的那几句话，自然就会有严重的信息失真。就像是我说 F=ma，就算说了 F是力，m是质量，a是加速度。有多少人分别能理解到多深呢？如果连基础科学都学不明白的人，就说这个东西是伪科学，恐怕是外行指导内行了。


一个人，它的一切大脑活动，可以被划分为：与人相关/与人无关；也可以划分为：与自己相关、与外界相关；按照对事物的理解，可以被划分为：抽象的和具体的。这分别是三个全集，且每种划分两两没有交集。

当然你如果说，还可以把别人再细分为：家人、朋友、同事、领导... 当然，那你可以整一个64人格对吧，可是，前者的分类也是没有问题的。

#### I/E

用I代指和倾向与自己相关，E代指倾向与与外界相关，没有问题，对吧。因为人总有种倾向，你如果说：我就是没有倾向，不偏不倚！或者说，凭什么你来定义这个边界？这些问题都合理，但是这种分类是可以存在的，这件事是不需要被质疑的。如何定边界决是关于测试题目和打分标准的问题，而不是理论框架本身。这点如果都没有想明白过，就评头论足属于瞎扯淡了。


#### S/N OR T/F

这两个容易和 T/F 混淆，因为 S 感觉 和 F 很像啊？ N 又和 T 很像。但他们是有很多根本上的不同之处的。


##### 第一个不同

- N 和 S 代表的是 感知。
- T 和 F 代表的是 判断。

但常被提到的解释是：N 和 S 是表示如何获取信息； T 和 F 是如何处理信息。 这个解释是错误的。因为，我们人类的传感器都是一样的，获取信息的方式也是一样的，视觉、听觉、触觉... 其实，他们都是表示处理处理信息。而真正的不同点是：

- N 和 S 是“被动”处理信息的方式(倾向)
- T 和 F 是“主动”处理信息的方式(倾向)

什么叫被动，以前我也很不理解。

直到 AI (LLM) ! LLM 有输入，有输出，但是我们不明白内部是如何把输入转换成输出的（注意我说的是不明白，不是不知道，人类尤其是相关工作者当然知道LLM的输入经过哪些张量运算和非线性计算得到的输出，但是我们不明白，不理解、不懂，为什么这样就能有这样的输出）；但是在LLM之前，我们传统算法都是可以被明白的，有的人学不明白，但是总有人能学明白，我们是这样那样一步一步拆解问题，然后解决问题的。这是可以被理解的。

虽然，人脑是一个巨大的 非线性的 LLM，somehow 我们有一种思维是主动类似传统算法在运作的，典型的区分你通过询问就可以得出来：比如别人问我， MBTI 你怎么想的？ 这篇文章就是我的回答，这是我主动的思考，有迹可循的且可以输出为人类语言的过程。而有另一种思维，我们可以理解为更像是 LLM 的 运作方式，其实多数子系统都是以这种方式运行的，比如我们见过一个人之后，就能从图片中认出来，怎么认出来的？都有双眼皮？都有大鼻子？都是黑皮肤？no 根本就没想这些，就认出来了！可以理解为 有一个专用的 VLM 睡觉的时候训练，白天负责推理，把识别结果告诉大脑负责“传统算法”的运算模块，用于后期分析处理。

所以从这一点去理解：就知道了所谓 脸盲，就是这个 “VLM” 模块没有发育好，所以它们只能用传统算法来做人脸识别，效果可想而知，咱们人类到现在也不行，最终甚至靠 VLM 模型自己也不知道咋识别的。从这一点还可以理解：所谓 ASD 谱系， 就是专用与分析识别 表情情绪 的 这个 专用神经网络(LLM的更泛化名称) 没有发育好。所谓高功能的阿斯伯格智商代偿，就是CPU和传统算法牛逼，识别效果堪比专用神经网络跑在GPU上。


##### 第二个不同

- N 和 S 代表的是 抽象和具体，而这个具体，可以是人也可以是物，抽象也可以人，也可以是物。
- T 和 F 代表的是 思维和情感，这两个都是抽象的，只不过情感是和人相关的抽象，而思维可以理解为和人不相关的抽象（不把人当人的抽象）。

结合上文所述的两大不同之处，其实就能发现 N/S 和 T/F 很多根本性的不同，对吧！ 所以这两个相互组合是完全不同的。

### 测试结果不稳定？

有很多人否定 MBTI 的理由是 它不稳定，测试结果会变。关于测试结果会变这件事得分四个方面去看。

1. 测试题目是否科学严谨？

2. 测试者是否能理解清楚题目

3. 测试者是否能尽量真实答题？


4. 会变的测试是否没有价值？

哪怕去三甲医院做检查，很多病的误诊率相当之高，这说明现代医学是伪科学吗？并不是；另一方面，人是会变的，比如静息心率，今年可能我是42，明显48，这说明心率测试不是科学？

关于心理学的研究经得起推敲的理论没有多少。


### 理论经不经的住检验？

首先，经不经的住检验也要辩证的看，如果只要是经不住实践检验的理论就被逐出科学，那么科学也没几个理论了，但实际上，严谨来说，过时的理论确实应该被“驱逐”，比如牛顿力学，但是我们还在学，且还在大量使用，为什么？

因为在实践的时候，我们需要对理论的计算量和方便理解的程度和其预测精度之间做取舍。所以我们一般不用相对论和量子力学计算单摆实验和弹簧振动，而当我们计算卫星GPS时，我们会用狭义相对论而不仅仅用牛顿力学来计算，因为误差已经大于我们需要的范围了。我们在计算电压电流时，一般不考虑用量子力学或者Maxwell Equation，虽然这明显是更正确解释电子/电场规律的理论。所以，我们不会在用“错误”的理论计算电压电流时被人说：你这是伪科学！因为实践早已正确你的理论是错的了，垃圾！

当然，这种现象不多，纯粹是因为物理和数学门槛比较高，挡住了很多二极管在这站队拉踩。比如计算机的门槛就更低一些，所以在计算机语言的选择上就会有明显的站队拉踩。其实计算机语言的使用和取舍，跟上文计算物理现象所使用理论的取舍有异曲同工之妙：如果你想要极好的的性能和极低的延迟，那就用C甚至汇编；如果你想要快速的开发速度而不那么在意性能，那就用typescript或者python；如果你想要权衡稳定性和性能的工业软件，那就用Qt；如果你想要不错性能又稳定的后端服务开发，那就用java或者go；真正理解计算机的人，不会在这拉踩哪门语言，而是根据自己项目的需要，学习并使用尽量合适的语言，如果你学会了一种语言还一知半解，怀着羞耻感带进了不擅长的领域，可能就会转化为站队拉踩，以建立脆弱的自信心避免羞耻。

而MBTI的严谨性的确远小于物理或者计算机，但以物理和计算机的严谨性来衡量心理学，本身也是一种非蠢即坏的拉踩罢了。因为人类对于人脑机理的研究严重不足，导致人脑的现象和物理结构存在巨大的理论鸿沟。但如何能区分人脑的现象呢？目前大概也只能用分类归纳的方式来总结了吧？总之我也想不出更好的方法。


### 理论有没有道理？

《理解 MBTI 的分类方式》一节就说明了还是相当有道理的对吧。但相比于物理、数学、计算机这种理工科，其 地基牢固程度、推理过程严谨程度、实践检验的准确度远远不及。但结合上文，这种对比也不公平对吧。


## MBTI 对自己有什么用

我认为 MBTI 对自己其实有很好的指导意义。首先，得能理解MBTI的定性的含义，还要能相对定量的理解自己的定位，然后你就可以根据你喜欢的和你擅长的，找到合适自己的领域，减少自己的内耗，让自己过的更快乐。没有MBTI能做到吗？当然能，如果足够了解自己的各个方面的细节、喜好、习惯等，只不过MBTI在一定程度上降低了这个事情的复杂度，当然也伴随着欠拟合（也就是有一定失真）。但不得不说这是一个相对快速大致更了解自己的工具。
